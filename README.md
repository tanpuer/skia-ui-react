# React Custom Renderer for SkiaUI

https://www.npmjs.com/package/react-reconciler?activeTab=readme
use **react-reconciler** to create custom react-renderer for SkiaUI.

1. npm install.
2. npm run build.
3. evaluate js in SkiaUI2 project.
